Welcome to All That Jazz!

All that Jazz was created for the musician, by a musician. Having noticed the struggle of carrying around the unwieldy, 500-page Real Book, Nirayka Monga decided to take matters into her own hands and built All That Jazz. She created it completely own her own, using HTML, CSS, and the JavaScript library JQuery. This web app provides access to 100+ of the most popular jazz standards, all on one simple website. Moreover, the sheets can be organized by categories such as genre, tempo, key, and time signature, simplifying the search process and allowing users to hone in on certain skills.
